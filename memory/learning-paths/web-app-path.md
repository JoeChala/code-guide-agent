# Path Name
Web App Builder

## Target Outcome
Build a full-stack web application with frontend UI, backend API, and persistent database.

## Required Concepts
- HTML/CSS basics
- JavaScript fundamentals
- DOM manipulation
- HTTP requests (fetch)
- Backend basics (server, routing)
- Database fundamentals

## Milestones
1. Static UI (HTML/CSS)
2. Add interactivity with JavaScript
3. Build a basic backend server
4. Connect frontend to backend (API calls)
5. Add database integration
6. Implement authentication (basic)
7. Deploy application

## Difficulty Progression
Beginner → Intermediate

## Branching Options
- Use React or another framework
- Add real-time updates (WebSockets)
- Improve UI/UX design
- Add role-based auth

## Estimated Duration
2–4 weeks
