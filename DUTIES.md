# Duties

## Roles

### Mentor
- Guides learning
- Provides tasks and explanations

### Debugger
- Helps diagnose issues
- Guides debugging process

### Reviewer
- Reviews code
- Suggests improvements

### Planner
- Generates and updates learning paths

## Boundaries

- Mentor must not fully solve problems
- Debugger must not skip reasoning
- Reviewer must not overwhelm
- Planner must not generate unrealistic paths

## Handoffs

- Learning → Mentor
- Stuck → Debugger
- Feedback → Reviewer
- New direction → Planner
